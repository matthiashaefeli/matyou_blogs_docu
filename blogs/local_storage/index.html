Local Storage

LocalStorage is a type of web storage that allows Javascript websites and apps to store and access data right in the browser with no expiration date.
This means the data stored in the browser will persist even after the browser window has been closed.

The setItem() method of the Storage interface, when passed a key name and value,
will add that key to the given Storage object, or update that key's value if it already exists.


localStorage.setItem('name', 'foo')


The getItem() method of the Storage interface, when passed a key name,
will return that key's value, or null if the key does not exist, in the given Storage object.


localStorage.getItem('name')
"foo"


The removeItem() method of the Storage interface, when passed a key name,
will remove that key from the given Storage object if it exists. If there is no item associated with the given key, this method will do nothing.


localStorage.removeItem('name')
localStorage.getItem('name')
null


The clear() method of the Storage interface clears all keys stored in a given Storage object.


localStorage.setItem('name', 'foo')
localStorage.getItem('name')
"foo"
localStorage.clear()
localStorage.getItem('name')
null


The key() method of the Storage interface, when passed a number n,
returns the name of the nth key in a given Storage object. The order of keys is user-agent defined, so you should not rely on it.


localStorage.setItem('name', 'foo')
localStorage.setItem('name2', 'bar')
localStorage.key(0)
'name'
localStorage.key(1)
'name2'


Do not store sensitive user information in localStorage!
LocalStorage is limited to 5MB across all major browsers
