React Native , Redux and firebase start.

How to set up a React Native App with Redux and firebase.

First we create the app with the command:

$ react-native init app_name

In my case the I create my app with Yarn.

Go into the created folder:

$ cd app_name

Now we add redux, react-redux, and firebase and start the app:

$ yarn add redux

$ yarn add react-redux

$ yarn add firebase

$ npx react-native run ios

Register the app in firebase:
