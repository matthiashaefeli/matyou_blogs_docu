# ASCII to char and back with ruby

```
2.7.0 :001 > 'A'.ord
 => 65
2.7.0 :002 >
```

```
2.7.0 :002 > 65.chr
 => "A"
2.7.0 :003 >
```

```
2.7.0 :009 > 'A'.bytes
 => [65]
2.7.0 :010 >
```

and what about Binary

```
2.7.0 :013 > 76.to_s(2)
 => "1001100"
2.7.0 :014 > "1001100".to_i(2)
 => 76
2.7.0 :015 >
```
