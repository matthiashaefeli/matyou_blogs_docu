# Ruby .chr and .ord

Returns a string containing the character represented by the int's value according to encoding
```
2.7.0 :001 > 97.chr
 => "a"
2.7.0 :002 > 210.chr
 => "\xD2"
 ```

 Returns the integer ordinal of one character string
 ```
 2.7.0 :003 > 'a'.ord
 => 97
2.7.0 :004 > ';'.ord
 => 59
 ```
